Project Name:
KeenKeeper – Relationship Management App

Short Description:
KeenKeeper is a modern web application that helps users manage and strengthen their personal and professional relationships. It allows users to track interactions, organize contacts, and gain insights into their social connections through an intuitive and visually engaging interface.

Technologies Used:
React.js,
Tailwind CSS & DaisyUI,
React Icons & Lucide React,
React Context API,
Recharts,
LocalStorage API,
React-Toastify.

Key Features:
1.Relationship Analytics Dashboard – Visualize interaction patterns and connection strength using charts.
2.Smart Timeline Filtering – Easily browse and filter past interactions for better tracking.
3.Networking & Collaboration Tools – Organize and manage contacts for both personal and professional growth.
